# PHASE7_SUMMARY.md

## What Phase VII Changed

Phase VII improved what the campaign actually measures, not just how it reports.

- The standard branch and refinement path now comes from a real persisted atlas generated by the background solver and exported through the bridge flow.
- Quantitative matching is now environment-aware, so the scorecard says which environment supplied each result instead of silently taking the first matching record.
- Sidecar evidence now declares its origin. In the standard run, coupling consistency is upstream-sourced and the other three sidecar channels are bridge-derived rather than heuristic.
- A linked Shiab companion run now exists for `first-order-curvature`, so the repository no longer has only identity-shiab evidence.

## What The Upgraded Campaign Tested

The upgraded standard campaign tested whether the repository still shows:

1. stable branch and refinement behavior when the Phase V bridge is fed by a real persisted atlas,
2. explicit sidecar coverage with source-origin labels,
3. clear separation between toy controls, stronger internal benchmarks, and falsifying side evidence,
4. environment-specific quantitative matching across toy, structured, and imported-style records.

The linked companion run then checked whether the branch and refinement conclusions survive a swap from `identity-shiab` to `first-order-curvature` on the same toy-control atlas path.

## What Now Counts As Stronger Evidence

- The bridge path is stronger because it now references real persisted state artifacts instead of a synthetic checked-in atlas.
- The scorecard is stronger because the failing imported-tier benchmark is tied to `env-imported-synthetic-example` explicitly in the artifacts.
- The sidecar path is stronger because the standard run no longer hides heuristic placeholder records as if they were upstream evidence.

## What Still Does Not Count As Real-World Evidence

- The standard atlas is real and persisted, but it is still a trivial zero-seed residual-inspection control run on toy geometry.
- The imported environment is still synthetic-example provenance with fabricated dataset metadata.
- The harder quantitative benchmark is still an internal benchmark, not an outside experiment or external dataset.
- The Shiab companion run broadens scope only partially because it uses the same trivial control atlas and is not yet part of the main standard campaign.

## What A Project Lead Can Decide Next

- Keep the current Phase VII campaign as the standard repository evidence path.
- Do not present the current outputs as physical validation or as real imported-data validation.
- Put Phase VIII effort into nontrivial upstream solves, genuine imported data, upstream observation sidecars, candidate-specific claim gating, and a nontrivial non-identity Shiab path.
