# SO33-BF-GRAVITY-ELECTROWEAK

## Source

- Title: Gravity and electroweak sector from symmetry breaking of an
  SO(3,3) BF theory
- Authors: P. Samuel Wesley, Tejinder P. Singh, J.M. Isidro
- Identifier: arXiv:2602.19151 (submitted 2026-02-22, revised 2026-03-30)
- Type: external primary preprint (not GU-native)

## Content Summary

An SO(3,3) BF-type gauge theory on six-dimensional split-signature
spacetime with MacDowell-Mansouri-type breaking to SU(2) x SU(2), yielding
two overlapping four-dimensional Lorentzian sectors. The left sector
realizes SU(2)_L x U(1)_Y -> U(1)_EM "through the standard Higgs
mechanism"; the right sector breaks SU(2)_R x U(1)_Y -> U(1)_dem with
gravity on the opposite-signature sector.
