# Implementation P311: Completion Observed-Sector W/Z Row-Selector Audit

Phase311 tests a narrow post-Phase310 loophole: whether the latest local
completion revision's observed-sector recovery program can promote the
Phase307 target-independent decoupled W/Z row selector into a physical
electroweak row law.

The audit reads
`TheoryCompletitionRevisions/Geometric_Unity_Completion_Reorganized_Updated_v29.md`
and records the lines where the revision supplies observed-sector governance,
phenomenological-mapping discipline, representation-decomposition intent,
typed observable-map requirements, and open proof obligations. It then checks
that governance against the missing physical row-map fields from Phase255,
Phase257, Phase295, Phase307, and Phase310:

- canonical physical W/Z row selector;
- photon/W/Z eigenstate projection rows;
- physical W/Z observable map;
- electroweak gauge embedding and quadratic mass-operator IDs for the selected
  Phase307 rows;
- branch-stable target-blind observed projection sidecars.

Result: the completion draft provides useful governance for observed-sector
recovery, but it does not provide the missing physical W/Z row-selector
theorem or photon/W/Z projection rows. Phase311 therefore keeps
`canFillPhase201WzContract=false` and does not promote W/Z masses.
