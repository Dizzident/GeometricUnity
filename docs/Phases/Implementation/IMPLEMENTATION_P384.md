# Implementation P384: Phase307 Basis-Energy Response-Image Proxy Audit

Phase384 tests whether Phase27 per-candidate canonical `SU(2)` basis-energy
fractions provide a finer target-independent proxy that can repair the
Phase381/382/383 suppressed-axis conflict for the current Phase307 selector
space.

Inputs:

- Phase27 charge-sector convention and per-candidate basis-energy metadata.
- Phase213 boson source-lineage blocker matrix.
- Phase307 target-independent decoupled W/Z row-selection law audit.
- Phase379 response-image carrier-axis characterization.
- Phase381 Phase302/307 response-image selector compatibility audit.
- Phase382 response-image observed-projection requirement audit.
- Phase383 Phase307 suppressed-axis counterfactual selector audit.

Behavior:

- Average Phase27 basis-energy fractions over each Phase307 W and Z source
  definition's candidate set.
- Measure each W definition's fraction on the Phase379-suppressed axis `1`.
- Count W definitions and predeclared selectors below a conservative
  low-suppressed-axis proxy threshold of `0.10`.
- Compare that proxy count with the much smaller Phase379 suppressed-projector
  fraction.
- Preserve the boundary that basis-energy fractions are internal metadata, not
  observed electroweak projection rows or physical response-image projectors.

Outcome:

- The minimum suppressed-axis basis-energy fraction among all `125` current
  Phase307 W definitions is `0.33627838840903185`.
- No W definition and no predeclared selector falls below the `0.10`
  low-suppressed-axis proxy threshold.
- The selected P302-scaled stable common law remains
  `p302-scaled-max-min-magnitude` with W definition
  `charged-ladder-all-axis-neutral-coherent-plus` and suppressed-axis fraction
  `0.44501616745604355`.
- Phase384 accepts zero Phase201/Phase256 contract fields and does not promote
  W/Z/H masses.

Boundary:

Phase384 is a proxy audit only. It does not prove that no future GU-native W
source theorem can use or avoid the suppressed carrier axis. It shows that the
currently persisted Phase27 basis-energy metadata and Phase307 selector space
do not repair the response-image sidecar conflict or fill any source-lineage
contract.
