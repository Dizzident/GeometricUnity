# Phase325 Electroweak Unitarity Scattering Source Audit

Phase325 audits the perturbative-unitarity / longitudinal W/Z scattering route.
This route is real electroweak physics: without a Higgs or equivalent EWSB
mechanism, longitudinal W/Z scattering loses perturbative unitarity at high
energy. It is not, by itself, a target-independent W/Z/H mass source.

The audit records the positive unitarity evidence, then keeps it separate from
promotion. A unitarity bound can motivate an EWSB sector and constrain broad
mass ranges, but the boson prediction package still needs GU-derived W/Z
absolute-scale rows, weak/hypercharge couplings or weak mixing, an electroweak
VEV, observed photon/W/Z/H projection rows, and a solved Higgs scalar
self-coupling/source lineage.

## Decision

Phase325 is expected to pass only as a negative boundary audit. Perturbative
unitarity is a consistency constraint and upper-bound argument, not an exact
GU-local source row for W, Z, or Higgs masses.
