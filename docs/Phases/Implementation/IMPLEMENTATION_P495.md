# Phase495 Implementation

Phase495 is the zero-physics A8 consumer for the exact Phase493 and Phase494
summary artifacts. Their live schemas, byte hashes, stable content-contract
hashes, and allowed precursor classifications are frozen in the implementation
before the findings are consumed.

The adjudicator uses conservative precedence: invalid precursor; supported
sampler implementation defect; structural estimator nonidentifiability;
insufficient statistics; invalid observable or channel; justified prospective
repair-pack design; and unresolved. Mixed or incomplete evidence falls toward
the earlier blocking or unresolved branches, never toward justification.

The result does not create or mutate the Phase481 preregistration pack and does
not reinterpret any Phase456 row. It has no authority to run sampling or
production, motivate Phase458 G3, satisfy Phase458 G5, fill a source contract,
or promote any result. A justified terminal means only that a future pack may
be designed after the independently binding Phase481 and written-authorization
gates.
