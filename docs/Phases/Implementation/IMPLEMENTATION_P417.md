# Implementation P417 - Vector-Spinor 144 Decomposition Probe

Phase417 follows the Phase416 restart directive: decompose the
source-pinned `Z_{1/2}` carrier's vector-spinor `144` remainder.

## Scope

The study constructs the gamma-trace map

```text
10 x 16+ -> 16-
```

using the same Cl(10), Pati-Salam/SM chain, and Sym^2 welded-spin embedding as
Phases407/411. The `144` is the kernel of that map. The phase then restricts
SM and welded generators to the kernel and records:

- the exact rank/kernel dimension check;
- the welded `(j1,j2)` content of the `144`;
- the internal SM block census for the `144`;
- the welded content of the chiral `2 x 144` carrier;
- the fail-closed source boundary for bosonic maps and observed-field rows.

## Result Boundary

Phase417 is not a prediction phase. The source-pinned carrier is fermionic, and
the current sources do not define an even-composite map or bosonic projection
that turns it into a scalar VEV carrier. Phase201 and Phase256 remain untouched.
