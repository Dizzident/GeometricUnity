# Implementation P165

P165 evaluates the two remaining boson unblocker routes after P164:

- derive a new W/Z bridge source;
- test source-level amplitude normalization.

The phase selects source-level amplitude normalization for the next attempt because P164 found no existing bridge-source candidate, while P118/P120 provide a concrete target-independent dimensional normalization candidate that can be tested without fitting W/Z targets.
