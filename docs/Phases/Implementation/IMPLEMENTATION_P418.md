# Phase418 Direction-Dependent Curvature/VEV Coupling Scan

Phase418 extends Phase410's uniform-curvature no-go by scanning the minimal
direction-dependent block-isotypic menu on the same `su(3)` control branch.

The phase verifies that the triplet/doublet/singlet block projectors commute
with the residual `su(2)+u(1)` action, then separates two cases:

- pure direction-dependent quadratic curvature terms still run away on the
  Phase405/410 rank-1 flat rays;
- stabilized Landau-style block terms can select the doublet, but only after
  adding a block mass law and quartic stabilizer that current sources do not
  define.

The result is fail-closed: it identifies the exact source-law shape still
needed for a curvature/VEV route, but it supplies no source-defined coupling,
VEV scale, observed-field projection, weak-angle lineage, pole extraction, or
GeV normalization.
