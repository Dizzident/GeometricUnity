# Phase422: Vector-Spinor 144 Bilinear Scalar-Capacity Audit

Phase422 extends the Phase416/417 source-pinned `Z_{1/2}` vector-spinor
branch by one representation-theory step. It reads Phase417's audited welded
content for the chiral `2 x 144` carriers and computes exact character
singlet capacity for the bilinear channels:

- `Z_L x Z_L`
- `Z_R x Z_R`
- `Z_L x Z_R`

The mixed-chirality Dirac-like channel has zero scalar capacity. The
same-chirality Majorana-like channels have scalar capacity (`264 + 264`), so
representation content alone does not rule out a bilinear vector-spinor scalar
ansatz.

The phase is fail-closed. It does not compute the full SM-stable subspace of
the 528-dimensional same-chirality scalar sector, and it supplies no
source-defined bosonic projection map, action, VEV selection, observed-field
rows, weak-angle lineage, pole extraction, or GeV normalization. Therefore it
does not fill Phase201 or Phase256 and promotes no physical boson masses.
