# Phase365 Dressing-Field Electroweak Observed-Variables Audit

Phase365 audits dressing-field, gauge-invariant Higgs-mechanism, and
SU(2)-invariant electroweak-variable sources as possible observed-field
extraction leads for W/Z/H prediction.

The phase records sources where gauge-invariant dressed variables are built
from gauge-variant fields, where dressing is distinguished from gauge fixing,
where electroweak variables are dressed by the Higgs field, and where
second-order Standard Model variables are made SU(2)-invariant.

The audit is intentionally non-promotional. The sources do not supply a
GU-native dressing field, source-derived observed-sector vacuum, electroweak
embedding, photon/W/Z/H projection rows, separate W/Z source rows, weak-angle
or gauge-coupling lineage, Higgs scalar-source and self-coupling lineage,
physical-pole extraction, a mass-scale source, or GeV normalization.

- `studies/phase365_dressing_field_electroweak_observed_variables_audit_001/output/dressing_field_electroweak_observed_variables_audit_summary.json`
