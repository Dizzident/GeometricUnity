# Phase 247: Direct Bridge Repairability Audit

## Purpose

Audit whether the existing P190/P191 direct W/Z bridge candidate can be repaired into a promotable W/Z absolute-mass prediction by wiring current registry data, rather than requiring new source evidence.

## Inputs

- Phase12 geometry, sibling background manifests, boson registry, and unified particle registry.
- Phase190 direct target-independent bridge-source law candidate.
- Phase191 direct bridge prediction decision.
- Phase206 direct bridge normalization closure.
- Phase213 blocker matrix.
- Phase245 minimal unlock contract.
- Phase246 minimal unlock candidate inventory.

## Result

The direct bridge route is not fixable from current registry artifacts. P190 has a stable target-independent candidate, but P191 still lacks theorem promotion, raw-amplitude gate closure, and particle-specific W/Z source rows. The current Phase12 registries expose only ambiguous `C0_NumericalMode` unified boson candidates, not observed W/Z source-lineage rows.
