# Implementation P228: Boson Mass Matrix Extraction Obstruction Audit

P228 distinguishes Hessian-like numerical artifacts from a physical W/Z/H mass matrix extraction.

The Standard Model obtains W/Z/H tree-level masses after electroweak symmetry breaking from the Higgs VEV and the quadratic terms in the bosonic Lagrangian. A GU analog must therefore provide:

- a source-derived vacuum or background solution,
- a vacuum-selection rule,
- a gauge-fixed or constraint-consistent quadratic expansion,
- a physical boson mass matrix,
- W/Z/photon/Higgs eigenstate projection,
- unit normalization to physical masses,
- filled source-lineage rows for the existing gates.

The repo has FullHessian-style source spectra, especially around Phase46, but those are mode/source diagnostics rather than a physical mass matrix around a target-independent GU vacuum. P228 keeps that distinction explicit and fail-closed.
