# Phase378 Full Connection-Carrier Shell-Response Gram Audit

Phase378 extends Phase377 from the persisted selected source-mode subspace to
the full `156`-coordinate connection-`1`-form carrier used by the Phase12 toy
backgrounds.

The audited object is still discrete-only:

```text
G_i = Psi_shell^dagger deltaK[e_i] Psi_shell
Q_ij = Re Tr(G_i^dagger G_j)
```

The study also reconstructs the Phase376 selected source-mode blocks as
linear combinations of the full coordinate blocks. This checks whether the
Phase377 selected-source response is the restriction of the full carrier map.

Run:

```bash
dotnet run --project studies/phase378_full_connection_carrier_shell_response_gram_audit_001
```

This is not a physical mass operator, action Hessian, gauge-reduced observed
field map, Higgs source, or GeV-normalized W/Z/H prediction.
