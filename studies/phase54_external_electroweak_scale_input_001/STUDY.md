# Phase LIV - External Electroweak Scale Input

Phase LIII identified the external disjoint scale lane as one admissible path.
Phase LIV ingests that lane using the CODATA 2022 Fermi coupling constant from
NIST and derives the electroweak vacuum scale
`v = 246.21965079413738 GeV`.

This still does not produce absolute W/Z mass predictions. The missing piece is
an internal bridge from the external electroweak scale to the repository's
`internal-mass-unit`. The phase records why existing artifacts are insufficient:

- W/Z physical mode records provide internal masses but no GeV bridge;
- Phase XII coupling proxies are not validated physical weak couplings;
- Phase XXXIII normalization is internal and dimensionless.

The next phase should derive or validate an internal electroweak bridge
observable. Only after that bridge exists can the external scale become a
GeV-per-internal-mass-unit calibration.
