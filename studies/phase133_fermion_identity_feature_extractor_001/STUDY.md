# Phase133 Fermion Identity Feature Extractor

This study materializes target-blind fermion identity feature records for the P131 coverage-repaired rows.

The extractor joins:

- P131 coverage-repaired sector-label rows
- P127 gauge-basis content diagnostics
- P128 SU(2) generator-sector diagnostics
- P132 derivation-source gate

It does not assign physical sector labels unless a diagnostic is promotable. Current P127/P128 diagnostics are mixed, so the output remains fail-closed with partial feature records.
