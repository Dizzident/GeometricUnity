# Phase 203 - Defensible Boson Value Manifest

This phase publishes the current boson value status as a stable manifest.

Only rows with `status=defensible-promoted` are scientific values. Failed and blocked rows are retained as explicit non-values so they cannot be accidentally promoted.

Current defensible values:

- W/Z mass ratio;
- photon protected masslessness;
- gluon protected masslessness.

Current non-values:

- W absolute mass;
- Z absolute mass;
- Higgs mass.
