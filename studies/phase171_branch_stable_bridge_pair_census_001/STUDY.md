# Phase 171 Branch-Stable Bridge Pair Census

This phase broadens the W/Z bridge-source search from the fixed Phase95 source pair to every promoted Phase91 branch-stable mode pair.

The census keeps the pair identity target-independent by matching mode-index pairs across the two sibling Phase12 backgrounds. It ranks internal raw matrix-element amplitudes, checks background stability, and only reports whether any source clears the existing raw-amplitude gate. It does not fit W/Z target residuals or promote a prediction without a later identity audit.
