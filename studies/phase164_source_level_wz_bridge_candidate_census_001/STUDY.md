# Phase 164 Source-Level W/Z Bridge Candidate Census

This phase checks whether any existing Phase12 fermion variation matrix can serve as a source-level W/Z bridge before another absolute-mass projection is attempted.

The census is diagnostic. It uses the fixed Phase91 source-selected fermion pair, applies each existing variation matrix under the same Euclidean mode normalization used by P163, and compares the raw matrix-element magnitude to the already declared P110 raw-amplitude gate. It does not fit a scale from W/Z target residuals and does not promote a bridge by nearest-target selection.
