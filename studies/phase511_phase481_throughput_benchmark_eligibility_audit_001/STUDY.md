# Phase511: Phase481 Throughput-Benchmark Eligibility Audit

Phase511 is the first Amendment A15 phase. It exact-binds the prospective
Phase481 plan, the frozen calibration and cost contracts, the reduced
anisotropic control result, and the historical Phase452 implementation source.

The phase asks a prerequisite question before any timing: does a unique,
source-bound production default and complete operation/measurement ledger
already exist for a production-equivalent throughput experiment? Current
Phase452 behavior is historical implementation evidence only; it is not
silently adopted as the future Phase481 default.

The audit runs no benchmark, HMC, sampling, reprocessing, pack construction,
full T16/T32 allocation, loader, or checkpoint integration. Any missing field
refuses before timing. It supplies no Phase458 input or physical-unit result,
and `promotedPhysicalMassClaimCount=0`.
