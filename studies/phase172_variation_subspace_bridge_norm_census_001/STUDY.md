# Phase 172 Variation-Subspace Bridge Norm Census

This phase tests whether the W/Z bridge source exists as a stable target-independent subspace across all Phase12 variation candidates rather than as one individual variation matrix.

For each ordered Phase91 mode-index pair, the phase computes the Euclidean norm of the complex matrix elements over the full variation-candidate basis on both sibling Phase12 backgrounds. It then applies the same raw-amplitude and sibling-stability gates used by the preceding bridge-source censuses. It does not fit any coefficient vector to W/Z targets.
