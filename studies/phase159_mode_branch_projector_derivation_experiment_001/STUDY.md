# Phase 159: Mode-Branch Projector Derivation Experiment

This phase attempts a target-independent calculation for the current boson blocker.

The experiment builds reduced Hermitian density matrices from the repaired fermion eigenvectors and diagonalizes their gauge-sector content. A promotable result would mean the repaired mode itself selects a stable branch projector without using W/Z targets or hand-assigned Pati-Salam labels.

If the density spectra are mixed, the phase records that a projector-derived mode-to-branch map cannot be obtained from the current repaired modes.
