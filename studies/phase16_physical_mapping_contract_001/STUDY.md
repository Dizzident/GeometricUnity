# Phase XVI Physical Mapping Contract Study

## Purpose

This study records candidate bridges from current benchmark observables to
physical boson properties. None are validated in Phase XVI. Each candidate is
blocked until the mapping, scale calibration, and target evidence are explicit.

## Candidate Mappings

| Candidate | Status | Current source | Closure requirement |
| --- | --- | --- | --- |
| photon masslessness / gauge mode identification | blocked | `bosonic-eigenvalue-ratio-1` as a possible low-mode proxy | Derive and test a protected gauge-mode identification map; show the quantity is not merely a benchmark eigenvalue ratio. |
| W boson charged weak-vector mass | blocked | no validated source observable | Add a charged vector-mode classifier, scale setting to GeV, and uncertainty propagation. |
| Z boson neutral weak-vector mass | blocked | no validated source observable | Add a neutral vector-mode classifier, electroweak mixing contract, scale setting, and tests. |
| Higgs scalar excitation mass | blocked | no validated source observable | Add a scalar excitation classifier and calibrated mass extraction path. |
| electroweak mixing angle | blocked | no validated source observable | Define coupling/mixing observables, normalization, and a dimensionless target comparison contract. |

## Phase XVII Inputs

- Add scale-setting artifacts before any mass-energy comparison.
- Add unit conversion and uncertainty propagation from internal spectral units.
- Keep physical target comparison fail-closed until mappings are validated and
  calibrated.

## Phase XVII Progress

The campaign now accepts an explicit physical calibration table. The checked-in
reference campaign contains only a blocked placeholder calibration, so the
physical claim gate remains closed. This is intentional: even dimensionless
claims such as photon masslessness require a declared normalization observable
and uncertainty propagation path before comparison with physical targets.
