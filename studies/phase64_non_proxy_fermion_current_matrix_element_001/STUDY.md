# Phase LXIV Non-Proxy Fermion-Current Matrix Element Source

## Purpose

Provide the replayable non-proxy operator matrix-element source needed before a normalized weak-coupling amplitude can be extracted.

## Result

The source is derived as `<phi_i, delta_D[b_k] phi_j>`, with `delta_D[b_k]` supplied by the analytical Dirac variation path.

The artifact explicitly records that it does not use finite-difference coupling proxies or coupling proxy magnitudes.

## Scope

This is a source derivation, not a weak-coupling value. The next phase must turn the matrix element into a dimensionless amplitude candidate and then supply uncertainty and branch-stability evidence before the Phase LXI gate can accept it.
