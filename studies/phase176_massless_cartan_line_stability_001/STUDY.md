# Phase 176 Massless Cartan-Line Stability

This phase checks whether the protected massless subspace contains a target-independent one-dimensional Cartan-like gauge direction that could support a photon-style U(1) identity.

It computes the gauge-component covariance of the curvature signatures for all massless modes on each sibling background, diagonalizes the covariance, and gates whether a dominant principal gauge line is both well separated and stable across backgrounds. It does not use photon or gluon targets.
