# Phase CXVIII - W/Z Matrix-Element Normalization Diagnostic

Diagnose the upstream normalization/operator-scale blocker left by Phase117.

This phase compares the repaired W/Z matrix elements, the older invalid
ungauged W/Z replay, source vector norms, fermion vector norms, generator
normalization, and dimensional scale candidates. It does not apply a repair
factor. Its job is to identify which repair classes are still plausible and
which are ruled out by the existing artifacts.
