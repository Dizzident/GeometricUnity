# Phase379 Response-Image Carrier-Axis Characterization

Phase379 characterizes the positive rank-`3` carrier image found by Phase378.
It works only from the target-blind full-carrier response Gram matrices:

```text
Q_ij = Re Tr(G_i^dagger G_j)
```

The study computes the positive eigenspace in the `156 = 52 x 3` edge-major
connection carrier, reports gauge-axis and edge support, and measures
background-to-background carrier-image principal angles.

Run:

```bash
dotnet run --project studies/phase379_response_image_carrier_axis_characterization_001
```

This is a discrete image-characterization audit. It is not a W/Z source row,
not a physical observed-field projection, not a Higgs source, and not a boson
mass prediction.
